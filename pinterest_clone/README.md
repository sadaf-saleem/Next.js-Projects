# Pinterest Clone 📌

A Pinterest-inspired image sharing web application built with Next.js and Firebase.

## Tech Stack

- **Next.js** - React framework
- **Firebase** - Database & Authentication
- **Tailwind CSS** - Styling
- **NextAuth.js** - Google Authentication
- **Cloudinary** - Image storage (Free)

## Features

- Google Sign In / Sign Out
- Create and save pins
- Image upload via Cloudinary
- View pins by user profile
- Responsive design
